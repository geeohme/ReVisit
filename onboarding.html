How do I get a key? (no terminal expertise needed)

Run this one-line command in your terminal; it returns a key starting with kb-llm-:

curl -X POST https://llmproxy.api.sparkbright.me/admin/apps \
  -H "Content-Type: application/json" \
  -d '{"appName": "ReVisit Extension", "rateLimit": 120}'

Copy the apiKey value and paste it above. Save it — it can’t be retrieved later. Your key is stored locally.
