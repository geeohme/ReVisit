🔑 LLM Gateway Setup

Your LLM Gateway API key provides access to 16+ AI providers through a unified interface.

How to Generate an LLM Gateway API Key

  1. Open your terminal or command prompt
  2. Copy and paste the following curl command:
curl -X POST https://llmproxy.api.sparkbright.me/admin/apps \
  -H "Content-Type: application/json" \
  -d '{"appName": "ReVisit Extension", "rateLimit": 120}'
  1. Press Enter to execute the command
  2. Copy the "apiKey" value from the response (starts with "kb-llm-")
  3. Paste it into the API Key field above
  4. Click "Test Connection" to verify it works

Note: Save your API key immediately - it cannot be retrieved later!
